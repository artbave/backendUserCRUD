# Users API (Spring Boot) — CRUD + Login JWT (para Retrofit)

Backend educativo en **Java 21** con **Spring Boot**, **Spring Security**, **JPA/H2**, **JWT**, **Lombok** y **MapStruct**.

Incluye:
- Registro de usuarios
- Login que devuelve **token JWT**
- Usuario **ADMIN por defecto**
- Endpoints de administración protegidos: listar/editar/borrar usuarios (solo ADMIN)

---

## 1) Requisitos

### Software
- **JDK 21** instalado
- **Maven 3.9+**
- (Opcional) **IntelliJ IDEA**
